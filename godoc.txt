PACKAGE

package stringset
    import "."

    Package stringset implements a very simple Set implementation for
    strings with a map[string]bool.

TYPES

type Set map[string]bool
    Set contains a set of strings.

func New() Set
    Create a new Set.

func (s Set) Add(v string)
    Add a value to a set.

func (s Set) Difference(t Set)
    Difference will do an in-place difference of t from s.

func (s Set) DifferenceL(t []string)
    DifferenceL will do an in-place difference of t from s, where t is a
    slice of strings.

func (s Set) In(v string) bool
    Test whether a value is in a set.

func (s Set) Intersection(t Set)
    Intersection will do an in-place intersection of t and s.

func (s Set) IntersectionL(t []string)
    IntersectionL will do an in-place intersection of t and s, where t is a
    slice of strings.

func (s Set) Remove(v string)
    Remove a value from a set.

func (s Set) Union(t Set)
    Union will do an in-place union of s and t.

func (s Set) UnionL(t []string)
    UnionL will do an in-place union of s and t, where t is a slice of
    strings.
